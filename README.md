## 1. Setup Project

### Clone the repository:

```bash
git clone https://github.com/andri-egils/quantum-dice-ed.git
cd quantum-dice-ed
```

---

# **2️⃣ Backend README.md**

`backend/README.md`

```markdown
# Quantum Dice Backend

FastAPI + Qiskit backend for the Quantum Dice project.
```
---

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate   # macOS/Linux
# Windows: .\.venv\Scripts\activate
pip install -r requirements.txt
```
